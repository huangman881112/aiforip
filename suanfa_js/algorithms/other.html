更多算法分类

这里收录了其他重要的算法类别，包括分治算法、数学与数论算法、字符串算法、机器学习算法、密码学算法、图像处理算法等。

分治算法

分治算法是将一个复杂问题分解为多个相似的子问题，递归解决子问题，然后合并子问题的解得到原问题解的算法设计策略。

归并排序

中等 排序

归并排序是一种高效的排序算法，采用分治策略，将数组分成两半，递归排序每一半，然后合并两个有序数组。

O(n log n)

快速排序

中等 排序

快速排序是一种高效的排序算法，采用分治策略，选择一个基准元素，将数组分为小于基准和大于基准的两部分，递归排序这两部分。

O(n log n) 平均

最近点对问题

较难 几何问题

最近点对问题是在平面上找出距离最近的一对点，使用分治策略可以将时间复杂度优化到O(n log n)。

O(n log n)

数学与数论算法

数学与数论算法是解决数学问题和数论问题的算法，包括最大公约数、素数判断、快速幂等经典算法。

欧几里得算法

简单 数论

欧几里得算法用于计算两个非负整数的最大公约数，基于辗转相除法的原理。

O(log min(a,b))

埃拉托色尼筛法

中等 数论

埃拉托色尼筛法是一种高效的素数筛选算法，用于找出小于等于给定整数n的所有素数。

O(n log log n)

快速幂

中等 数论

快速幂算法用于高效计算幂运算，将指数分解为二进制形式，利用平方的性质减少乘法次数。

O(log n)

字符串算法

字符串算法是处理字符串数据的算法，包括字符串匹配、字符串查找、字符串处理等操作。

KMP算法

中等 字符串匹配

KMP算法是一种高效的字符串匹配算法，通过预处理模式串构建失效函数，避免不必要的字符比较。

O(n+m)

Rabin-Karp算法

中等 字符串匹配

Rabin-Karp算法是一种基于哈希的字符串匹配算法，通过计算子串的哈希值来快速比较字符串。

O(n+m) 平均

Z-Algorithm

中等 字符串处理

Z-Algorithm用于构建一个数组，其中每个元素表示字符串前缀与该位置开始的子串的最长公共前缀长度。

O(n)

机器学习算法

机器学习算法是让计算机从数据中学习规律并做出预测的算法，包括监督学习、无监督学习和深度学习等类别。

线性回归

中等 监督学习

线性回归是一种用于建模自变量和因变量之间线性关系的监督学习算法，广泛应用于预测和回归分析。

O(n)

决策树

中等 监督学习

决策树是一种树形结构的分类算法，通过一系列的判断条件将数据划分到不同的类别中。

O(n log n)

K-Means聚类

中等 无监督学习

K-Means聚类是一种无监督学习算法，用于将数据划分为K个不同的簇，每个簇内的数据点尽可能相似。

O(nkt)，n是数据量，k是簇数，t是迭代次数

密码学算法

密码学算法是用于保护信息安全的算法，包括加密算法、解密算法、哈希算法等，广泛应用于网络安全和数据保护。

AES加密

较难 对称加密

AES（高级加密标准）是一种对称加密算法，被广泛应用于数据加密保护，支持128位、192位和256位密钥长度。

O(n)

RSA加密

较难 非对称加密

RSA是一种非对称加密算法，使用公钥加密、私钥解密，广泛应用于安全通信和数字签名。

O(n³) 密钥生成，O(n²) 加密和解密

SHA-256哈希

中等 哈希算法

SHA-256是一种密码学哈希函数，能够产生一个256位的哈希值，广泛应用于数据完整性校验和密码存储。

O(n)

图像处理算法

图像处理算法是用于处理和分析图像的算法，包括图像增强、图像分割、边缘检测、特征提取等操作。

Canny边缘检测

中等 边缘检测

Canny边缘检测是一种多级边缘检测算法，能够检测出图像中的边缘，并抑制噪声，广泛应用于计算机视觉领域。

O(nm)

直方图均衡化

中等 图像增强

直方图均衡化是一种图像增强技术，通过调整图像的灰度直方图，提高图像的对比度和清晰度。

O(nm)

模板匹配

中等 图像识别

模板匹配是一种在图像中寻找与给定模板相似区域的技术，广泛应用于目标检测和图像识别。

O((n-w+1)(m-h+1)wh)

回溯算法

回溯算法是一种通过探索所有可能的候选解来找出所有解的算法，当发现候选解不满足条件时，回溯到上一步继续探索。

N皇后问题

中等 经典问题

N皇后问题是将N个皇后放置在N×N的棋盘上，使得它们互不攻击，即任意两个皇后不能处于同一行、同一列或同一斜线上。

O(N!)

数独求解

中等 约束满足

数独求解是填充9×9的网格，使得每行、每列和每个3×3子网格都包含1-9的数字，没有重复。

O(9^m)，m是空格数量

子集和问题

中等 组合优化

子集和问题是找出一个集合的子集，使得子集中元素的和等于给定的目标值，是一个NP完全问题。

O(2^n)
